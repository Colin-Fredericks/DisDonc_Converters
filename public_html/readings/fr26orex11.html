EXERCICES ORAUX 11–12

Exercice oral 11: Tout le monde

Answer the following questions affirmatively, saying that the people mentioned do the things the speaker asks about.

Modèle

Les enfants se sont endormis?

Oui, ils se sont tous endormis.

Commençons.

Exercice oral 12: Sans exception

Answer the following questions affirmatively. Note that tous and toutes refer to the objects of the verb.

Modèle

Est-ce que tu as ouvert les fichiers?

Oui, je les ai tous ouverts.

Commençons.
