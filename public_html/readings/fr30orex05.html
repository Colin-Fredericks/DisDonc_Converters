EXERCICES ORAUX 5–7

Exercice oral 5: Moi, je l’aurais fait

Say that you would have done all the things that Philippe didn’t do. Replace object nouns with the corresponding pronouns.

Modèle

Philippe n’avait pas commandé les billets.

Moi, je les aurais commandés.

Commençons.

Exercice oral 6: Ce qu’on aurait dû faire

Say what people should have done, using the conditional perfect of devoir. Replace object nouns with the corresponding pronouns in your answers.

Modèle

Karim n’a pas fini le projet.

Il aurait dû le finir.

Commençons.

Exercice oral 7: Et vous?

The speaker tells you things people did not complete. Ask the speaker if he/she would have done them. Replace object nouns with the corresponding pronouns in your answers.

Modèle

M. Jolivet n’a pas embauché ces informaticiens.

Et vous, est-ce que vous les auriez embauchés?

Commençons.
