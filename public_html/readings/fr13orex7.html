EXERCICES ORAUX 7-8

Exercice oral 7: On doit!

Using the verb devoir, say what the following people must do.

Modèle

Christine ne peut pas travailler.

Mais elle doit travailler.

Commençons.

Exercice oral 8: Vouloir n'est pas toujours pouvoir

The following people want to do something, but can't. Say so, using both vouloir and pouvoir.

Modèle

Nicole / retrouver ses amis

Nicole veut retrouver ses amis, mais elle ne peut pas.

Commençons.
