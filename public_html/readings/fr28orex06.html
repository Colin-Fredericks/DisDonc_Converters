EXERCICES ORAUX 6–7

Exercice oral 6: Pas un

Use aucun/aucune to answer the following questions.

Modèle

Il reste des pommes?

Non, il ne reste aucune pomme.

Commençons.

Exercice oral 7: Introuvable

None of the following people or things could be found anywhere. Say so, using nulle part. Replace all object nouns with the appropriate object pronoun.

Modèle

Tu as vu Nicole?

Non, je ne l’ai vue nulle part.

Commençons.
