Exercice écrit 2

Using demonstrative pronouns, complete the following dialogue.

Pardon, Monsieur. Je voudrais acheter un rasoir. / ____ est un très bon rasoir.

Tu aimes cette voiture? Je la trouve splendide. / Oui, elle est magnifique, mais _____ est encore plus merveilleuse.

Cette valise est assez grande? / Elle est trop grande. _____ est plus pratique.

Toutes ces sacoches sont pleines, n’est-ce pas? / Pas toutes. _____ sont vides.

Les cédéroms que j’ai ne sont pas de bonne qualité. / Écoutez _____. Ils sont de meilleure qualité.

Il me faut une savonnette. / Achète _____. Elle est très douce.

Quelle autoroute faut-il prendre? / _____ va directement à Lyon.

Quels bonbons est-ce que je dois acheter? / J’achète toujours _____. Ils sont vraiment délicieux.

Tu aimes ces cartes postales? / Oui, mais _____ sont plus intéressantes.

Je vais acheter ces tennis. / Attends. _____ sont meilleures.
