WRITING EXERCISE 1

Assume that the people or things mentioned in each sentence have the same characteristic as the original person or thing described.

Modèle

Marie-Thérèse est nerveuse.

Comment est son frère?

Il est nerveux aussi.

Tes cousins sont généreux. Comment est ta tante?

Leur appartement est affreux. Comment est leur maison?

Ces bonbons sont délicieux. Comment sont les tartes?

Le voisin est grincheux. Comment est la voisine?

Les étudiants sont paresseux. Comment sont les étudiantes?

Cette revue est ennuyeuse. Comment est ce journal?

Ces hommes sont indécis. Comment sont ces femmes?
