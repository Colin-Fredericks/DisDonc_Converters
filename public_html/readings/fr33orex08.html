EXERCICES ORAUX 8–10

 

Exercice oral 8: De celui-là

 

Use dont to answer the speaker’s questions.

 

Modèle

Vous parlez d’un article. Il est intéressant?

Oui, l’article dont nous parlons est très intéressant.

Commençons.

 

 

Exercice oral 9: Ah, celui-là

 

Use dont to identify the following people.

 

Modèle

Tu vois ce garçon-là? Son père est PDG de l’entreprise.

Ah, c’est le garçon dont le père est PDG de l’entreprise.

Commençons.

 

 

Exercice oral 10: C’est ça

 

Rephrase these sentences, using ce dont to highlight the object nouns.

 

Modèle

crayon / Tu as besoin d’un stylo, n’est-ce pas?

Non, ce dont j’ai besoin, c’est d’un crayon.

Commençons.
