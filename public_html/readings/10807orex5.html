ORAL EXERCISES 5-7

Oral Exercise 5: Intensification of Adjectives

Describe how interesting these things are by using the adverb indicated.

Modèle

ce cours/très

Ce cours est très intéressant.

Commençons.

Oral Exercise 6: Pas du tout

Contradict the speaker in each case using pas du tout.

Modèle

Marie est paresseuse, n'est-ce pas?

Mais non! Elle n'est pas paresseuse du tout.

Commençons.

Oral Exercise 7: Drôlement. vous êtes d'accord.

Agree with the speaker's statements by using drôlement.

Modèle

Ce logiciel est cher.

Drôlement cher.

Commençons.
