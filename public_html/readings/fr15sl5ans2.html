The above past participles are behaving like adjectives because they show agreement.

The following sentences contain object pronouns. Pay close attention to the gender and number of the noun and how the past participle reflects the changes in number and gender.

C’est pour ça que je l’ai choisi. The object pronoun l’ refers to logiciel and is masculine singular.

Les enfants l’ont mangée? The object pronoun l’ refers to tarte and is feminine singular.

Les messages? Je les ai effacés. The object pronoun les refers to messages and is masculine plural.

Où sont mes clés? Je les ai perdues! The object pronoun les refers to clés and is feminine plural.

The past participle agrees in gender and number with the preceding direct object pronoun.
