il met

he puts

mauvais

bad; wrong

dans le mauvais sac

une erreur

même

same

the same mistake that I made

Et il met son message dans le mauvais sac. La même erreur que moi! Alors, le message est pour moi.

And he puts his message in the wrong bag. The same error that I made. So, the message is for me.
