EXERCICES ORAUX 12-13

Exercice oral 12: Oui. Faites-le.

Answer each of the following questions with a tu-command. Replace all object nouns with the corresponding pronouns.

Modèle

Tu veux que je te prête mon vélo?

Oui, prête-le-moi.

Commençons.

Exercice oral 13: Oui, c’est ce qu’on fera.

Answer each of the following questions with a nous-command. Replace all object nouns with the corresponding pronouns.

Modèle

Vous voulez qu’on prête nos vélos aux voisins?

Oui, prêtons-les-leur.

Commençons.
